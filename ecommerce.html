eCommerce: Bulk Grocery Orders

Summary

Project Directive

For customers on mobile devices that need to place bulk orders of dry goods and produce for restaurants and markets:

  1. Deliver a streamlined login experience.
  2. Create an intuitive search results page for users to add products to their order.
  3. Build filters and sort options for users to quickly locate content.
  4. Design a cart that allows users to duplicate past orders and edit current transactions.

My Contributions

As the lead UI designer, I established patterns and styles for all visual components and shared style documentation with developers.
